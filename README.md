## Постановка задачі

Розробити програму, за допомогою якої на основі **звичайних жорданових виключень (ЗЖВ)** можна реалізувати:

1. Пошук оберненої матриці для довільної квадратної матриці
2. Обчислення рангу довільної прямокутної матриці
3. Розв'язання СЛАР за допомогою оберненої матриці (1-й спосіб)

### Варіант 7

```
5x₁ -  x₂ -  x₃ = 5
3x₁ + 2x₂ -  x₃ = 6
 x₁ - 2x₂ - 3x₃ = 2
```

Матриця A та вектор B:

```
        | 5  -1  -1 |         | 5 |
A   =   | 3   2  -1 |   B  = | 6 |
        | 1  -2  -3 |         | 2 |
```

## Використаний алгоритм

Застосовано алгоритм **звичайних жорданових виключень**. Суть алгоритму — обирається розв'язувальний елемент у таблиці і за певними правилами перераховуються всі інші елементи.

На основі цього алгоритму реалізовано:

- **Обернена матриця** — виконання n процедур ЗЖВ по головній діагоналі
- **Ранг матриці** — підрахунок кількості процедур ЗЖВ, які вдалося виконати по діагоналі
- **Розв'язання СЛАР** — знаходження оберненої матриці та множення її на вектор B

### Алгоритм процедури ЗЖВ

```
1. Обрати розв'язувальний елемент a[r][c]
2. Якщо a[r][c] ≈ 0 → пропуск

3. Зберегти копію таблиці
4. a[r][c] ← 1

5. Розв'язувальний рядок (j ≠ c):
       a[r][j] ← −a_old[r][j]

6. Інші елементи (i ≠ r, j ≠ c):
       a[i][j] ← (a_old[i][j] · a_old[r][c] − a_old[i][c] · a_old[r][j])

7. Всю таблицю поділити на a_old[r][c]

8. Обміняти мітки: left[r] ↔ top[c]
```

## Протоколи обчислень

### 1. Пошук оберненої матриці

```
Початок:
                x1          x2          x3
y1          5,0000     -1,0000     -1,0000
y2          3,0000      2,0000     -1,0000
y3          1,0000     -2,0000     -3,0000

Крок 1 — Розв. ел. [0][0] = 5
                y1          x2          x3
x1          0,2000      0,2000      0,2000
y2          0,6000      2,6000     -0,4000
y3          0,2000     -1,8000     -2,8000

Крок 2 — Розв. ел. [1][1] = 2,6
                y1          y2          x3
x1          0,1538      0,0769      0,2308
x2         -0,2308      0,3846      0,1538
y3          0,6154     -0,6923     -3,0769

Крок 3 — Розв. ел. [2][2] = -3,0769
                y1          y2          y3
x1          0,2000      0,0250     -0,0750
x2         -0,2000      0,3500     -0,0500
x3          0,2000     -0,2250     -0,3250
```

**Результат A⁻¹:**

```
|  0,2000    0,0250   -0,0750 |
| -0,2000    0,3500   -0,0500 |
|  0,2000   -0,2250   -0,3250 |
```

### 2. Обчислення рангу матриці

Виконано ті самі 3 кроки ЗЖВ по діагоналі (див. вище).

**Ранг = 3**

### 3. Розв'язання СЛАР (1-й спосіб)

Спочатку знаходимо A⁻¹, потім обчислюємо **X = A⁻¹ · B**:

```
X = A⁻¹ · B

x1 =  0,2000·5 +  0,0250·6 + (-0,0750)·2 =  1
x2 = -0,2000·5 +  0,3500·6 + (-0,0500)·2 =  1
x3 =  0,2000·5 + (-0,2250)·6 + (-0,3250)·2 = -1
```

**Відповідь: x₁ = 1, x₂ = 1, x₃ = −1**
